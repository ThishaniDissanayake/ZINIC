 Setup Instructions ==>

### Frontend Setup
1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Open `index.html` in your preferred web browser.

### Backend Setup
1. Ensure you have PHP and MySQL installed on your local server.
2. Import the `zinic.sql` file located in the `Database` folder into your MySQL database.
3. Modify the database connection settings in the PHP files to match your local setup.
4. Add the "zinic_" folder to the htdocs in XAMPP folder 
5. Start your local server.

 Accessing the Website ==>
- Open your web browser and enter the following URL: `localhost/zinic_`
